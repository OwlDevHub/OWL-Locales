# owl-locales

Локализации для THE OWL PROJECT (en, ru, de, es, ja)

## Установка

```sh
yarn add <git-url>
# или
npm install <git-url>
```

## Использование

```js
import { en, ru, de, es, ja } from 'owl-locales';

const resources = {
  en: { translation: en },
  ru: { translation: ru },
  de: { translation: de },
  es: { translation: es },
  ja: { translation: ja },
};
```
