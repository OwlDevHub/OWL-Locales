# owl-locales

Localization for OWL

## Langs:

- en
- ru
- de
- es
- ja

## Install

```sh
yarn add https://github.com/OwlDevHub/OWL-Locales
# or
npm install https://github.com/OwlDevHub/OWL-Locales
```

## Usage

```js
import { en, ru, de, es, ja } from "owl-locales";

const resources = {
  en: { translation: en },
  ru: { translation: ru },
  de: { translation: de },
  es: { translation: es },
  ja: { translation: ja },
};
```
